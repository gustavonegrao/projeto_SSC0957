# projeto_SSC0957

INTRODUÇÃO 
Projeto da disciplina Prática em Ciência de Dados II (SSC0957) que consiste em utilizar dados públicos disponibilizados pela Fundação SEADE e pelo Portal da Transparência do Estado de São Paulo para analisar O mercado de trabalho formal, os fatores econômicos (PIB e IPCA por setor) e os dados do Cadunico.
